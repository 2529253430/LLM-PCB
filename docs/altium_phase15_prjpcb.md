# Phase 15: Native Altium `.PrjPcb` Project Shell

Phase 15 introduces the first native Altium Designer file generated by LLM-PCB.

## Output

```text
output/altium_phase15/Buck_24V_to_5V_3A/
├── Buck_24V_to_5V_3A.PrjPcb
├── altium_project_model.json
└── manifest.json
```

The `.PrjPcb` file is an ASCII Altium PCB project file. At this phase it is
intentionally a project shell and contains no registered design documents.
Phase 16 will generate and register the schematic document.

## Run tests

```powershell
python -m pytest -q test/test_altium_prjpcb_writer.py
python -m pytest -q
```

## Run example

```powershell
python -m examples.export_altium_prjpcb
```

## Manual validation

Open the generated `.PrjPcb` with `File > Open Project` in Altium Designer.
The project should appear in the Projects panel without SchDoc or PcbDoc.
